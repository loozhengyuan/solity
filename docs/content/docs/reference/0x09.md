title: 0x09 - Init Response
weight: 14

This packet is exclusively emitted by the Door Lock in response to the [`0x05`](../0x05) packet and is acknowledged by a [`0x89`](../0x89) packet.

## Specification

| Byte | Description                               | Values |
| ---- | ----------------------------------------- | ------ |
| 1    | Packet ID                                 | `0x09` |
| 2    | Unknown                                   | `0x01` |
| 3–18 | Device Identifier (ASCII, fixed 16 bytes) |

## Examples

```
FF 48 12 09 01 48 47 44 4C 37 30 30 32 38 35 54 57 4B 00 00 00 67
```
