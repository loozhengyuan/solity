---
title: 0x5A - Info Request
weight: 25
---

This packet is exclusively emitted by the Module to retrieve lock info and is acknowledged by a [`0xDA`](../0xDA) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x5A` |

## Examples

```
FF FF FF 48 01 5A 5B
```
