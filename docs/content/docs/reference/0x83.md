This packet is exclusively emitted by the Module in response to the [`0x03`](../0x03) packet.

## Specification

| Byte | Description | Values                                 |
| ---- | ----------- | -------------------------------------- |
| 1    | Packet ID   | `0x83`                                 |
| 2    | State?      | `0x00` - Unpaired?<br>`0x01` - Paired? |

## Examples

### When Paired

```
FF FF FF 48 02 83 01 86
```

### When Unpaired

```
FF FF FF 48 02 83 00 85
```
