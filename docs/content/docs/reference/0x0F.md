---
title: 0x0F - Module Init
weight: 16
---

This packet is exclusively emitted by the Module on power up regardless whether connected to the Door Lock or not. The packet is **never acknowledged**.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x0F` |
| 2    | Unknown     | `0x11` |
| 3    | Unknown     | `0x01` |
| 4    | Unknown     | `0x10` |
| 5    | Unknown     | `0x01` |

## Examples

```
FF FF FF 48 05 0F 11 01 10 01 37
```
