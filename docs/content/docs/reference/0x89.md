This packet is exclusively emitted by the Module in response to the [`0x09`](../0x09) packet.

## Specification

| Byte | Description          | Values |
| ---- | -------------------- | ------ |
| 1    | Packet ID            | `0x89` |
| 2-3  | Year (Little Endian) |        |
| 4    | Month                |        |
| 5    | Date                 |        |
| 6    | Hour                 |        |
| 7    | Minute               |        |
| 8    | Second               |        |
| 9    | Unknown              | `0x11` |
| 10   | Unknown              | `0x01` |

## Examples

```
FF FF FF 48 0A 89 EA 07 01 02 0F 23 3B 11 01 82
```

```
FF FF FF 48 0A 89 EA 07 01 02 0F 24 00 11 01 48
```

