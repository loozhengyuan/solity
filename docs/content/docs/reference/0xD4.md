---
title: 0xD4 - Lock Arm Event Ack
weight: 4
---

This packet is exclusively emitted by the Module in response to the [`0x54`](../0x54) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0xD4` |

## Examples

```
FF FF FF 48 01 D4 D5
```
