---
title: 0xA1 - Lock Disarm Response
weight: 10
---

This packet is exclusively emitted by the Door Lock in response to the [`0x21`](../0x21) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0xA1` |
| 2    | Unknown     | `0x00` |

## Examples

```
FF 48 02 A1 00 A3
```
