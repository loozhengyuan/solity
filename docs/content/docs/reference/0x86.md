---
title: 0x86 - Pair Success Event Ack
weight: 22
---

This packet is exclusively emitted by the Door Lock in response to the [`0x06`](../0x06) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x86` |

## Examples

```
FF 48 01 86 87
```
