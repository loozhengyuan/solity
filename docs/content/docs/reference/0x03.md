---
title: 0x03 - Registration Event
weight: 7
---

This packet is exclusively emitted by the Door Lock to indicate when the registration button is pressed and is acknowledged by a [`0x83`](../0x83) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x03` |

## Examples

```
FF 48 01 03 04
```
