---
title: 0x0A - Init Request
weight: 23
---

This packet is exclusively emitted by the Door Lock during initial configuration and is acknowledged by a [`0x8A`](../0x8A) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x0A` |

## Examples

```
FF 48 01 0A 0B
```
