This packet is exclusively emitted by the Door Lock in response to the [`0x51`](../0x51) packet.

## Specification

| Byte | Description | Values                              |
| ---- | ----------- | ----------------------------------- |
| 1    | Packet ID   | `0xD1`                              |
| 2    | Lock State  | `0x00` - Disarmed<br>`0x01` - Armed |
| 3    | Door State  | `0x00` - Opened<br>`0x01` - Closed  |
| 4    | Battery?    |                                     |
| 5    | Unknown     | `0x07`                              |
| 6    | Unknown     | `0x03`                              |
| 7    | Unknown     | `0x00`                              |
| 8    | Unknown     | `0x00`                              |
| 9    | Unknown     | `0x00`                              |
| 10   | Unknown     | `0x01`                              |

## Examples

### Lock Disarmed But Door Closed

```
FF 48 0A D1 00 01 53 07 03 00 00 00 01 3A
```

### Door Opened

```
FF 48 0A D1 00 00 45 07 03 00 00 00 01 2B
```

### Door Closed

**Battery 52%**

```
FF 48 0A D1 01 01 48 07 03 00 00 00 01 30
```

**Battery 49%**

```
FF 48 0A D1 01 01 45 07 03 00 00 00 01 2D
```
