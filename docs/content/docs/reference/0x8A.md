This packet is exclusively emitted by the Module in response to the [`0x0A`](../0x0A) packet.

## Specification

| Byte | Description          | Values |
| ---- | -------------------- | ------ |
| 1    | Packet ID            | `0x8A` |
| 2-3  | Year (Little Endian) |        |
| 4    | Month                |        |
| 5    | Date                 |        |
| 6    | Hour                 |        |
| 7    | Minute               |        |
| 8    | Second               |        |
| 9    | Unknown              | `0x11` |
| 10   | Unknown              | `0x01` |

## Examples

```
FF FF FF 48 0A 8A EA 07 01 0A 10 16 37 11 01 FF
```
