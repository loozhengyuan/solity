---
title: 0xD3 - Lock Disarm Event Ack
weight: 6
---

This packet is exclusively emitted by the Module in response to the [`0x53`](../0x53) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0xD3` |

## Examples

```
FF FF FF 48 01 D3 D4
```
