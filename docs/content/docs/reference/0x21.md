---
title: 0x21 - Lock Disarm Request
weight: 9
---

This packet is exclusively emitted by the Module to disarm lock and is acknowledged by a [`0xA1`](../0xA1) packet.

Based on observations, it is **only emitted when disarmed from App** and never when disarmed mechanically or via fingerprint.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x21` |
| 2    | Unknown     | `0x00` |
| 3    | Unknown     | `0x00` |
| 4    | Unknown     | `0x07` |

## Examples

```
FF FF FF 48 04 21 00 00 07 2C
```
