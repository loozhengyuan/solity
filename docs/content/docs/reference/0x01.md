---
title: 0x01 - Reset Request
weight: 18
---

This packet is exclusively emitted by the Door Lock when the pair/unpair sequence has been completed and is acknowledged by a [`0x81`](../0x81) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x01` |

## Examples

```
FF 48 01 01 02
```
