---
title: 0x51 - Status Request
weight: 11
---

This packet is exclusively emitted by the Module to retrieve lock status and is acknowledged by a [`0xD1`](../0xD1) packet.

Based on observations, this packet is triggered every 10s.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x51` |

## Examples

```
FF FF FF 48 01 51 52
```
