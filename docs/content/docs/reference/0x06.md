---
title: 0x06 - Pair Success Event
weight: 21
---

This packet is exclusively emitted by the Module to retrieve lock status and is acknowledged by a [`0x86`](../0x86) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x06` |

## Examples

```
FF FF FF 48 01 06 07
```
