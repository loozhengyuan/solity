This packet is exclusively emitted by the Module when the app is loaded and exited when connected to the Door Lock. The packet is **never acknowledged**.

## Specification

| Byte | Description          | Values                                        |
| ---- | -------------------- | --------------------------------------------- |
| 1    | Packet ID            | `0x64`                                        |
| 2-3  | Year (Little Endian) |                                               |
| 4    | Month                |                                               |
| 5    | Date                 |                                               |
| 6    | Hour                 |                                               |
| 7    | Minute               |                                               |
| 8    | Second               |                                               |
| 9    | State?               | `0x00` - Disconnected?<br>`0x01` - Connected? |
| 10   | State?               | `0x00` - Disconnected?<br>`0x01` - Connected? |
| 11   | Unknown              | `0x00`                                        |
| 12   | Unknown              | `0x00`                                        |
