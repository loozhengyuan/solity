---
title: 0x81 - Reset Response
weight: 20
---

This packet is exclusively emitted by the Module in response to the [`0x01`](../0x01) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x81` |

## Examples

```
FF FF FF 48 01 81 82
```
