---
title: 0x54 - Lock Arm Event
weight: 3
---

This packet is exclusively emitted by the Door Lock to indicate when it is armed and is acknowledged by a [`0xD4`](../0xD4) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0x54` |

## Examples

```
FF 48 01 54 55
```
