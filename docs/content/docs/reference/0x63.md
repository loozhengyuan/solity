---
title: 0x63 - Door State Event
weight: 1
---

This packet is exclusively emitted by the Door Lock to reflect the current door open/close state and is acknowledged by a [`0xE3`](../0xE3) packet.

## Specification

| Byte | Description | Values                             |
| ---- | ----------- | ---------------------------------- |
| 1    | Packet ID   | `0x63`                             |
| 2    | State       | `0x00` - Opened<br>`0x01` - Closed |

## Examples

### Door Opened

```
FF 48 02 63 00 65
```

### Door Closed

```
FF 48 02 63 01 66
```
