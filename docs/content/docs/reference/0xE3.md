---
title: 0xE3 - Door State Event Ack
weight: 2
---

This packet is exclusively emitted by the Module in response to the [`0x63`](../0x63) packet.

## Specification

| Byte | Description | Values |
| ---- | ----------- | ------ |
| 1    | Packet ID   | `0xE3` |

## Examples

```
FF FF FF 48 01 E3 E4
```
